## Optimization Guidelines

1. **Format**: Use WebP for best compression and quality
2. **Compression**: Aim for 80-90% quality to balance file size and visual fidelity
3. **Color Mode**: RGB for web
4. **Export from Figma**: Use "Export as WebP" or use a tool like Squoosh.app
5. **File Size Target**: Under 200KB per image
6. **Accessibility**: Ensure sufficient contrast and readable text in screenshots

## Alternative Formats (Optional)

Consider providing fallback formats for older browsers:
- PNG for transparency needs
- JPEG for photographic content

- [Squoosh](https://squoosh.app/) - Online image optimizer
